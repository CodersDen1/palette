
# Palette 2

## Overview
A modern color palette management tool for developers and designers.

## Features
- 🎨 Create and manage custom color palettes
- 📋 Export in multiple formats (JSON, CSS, SCSS)
- 🔍 Color picker and accessibility checker
- 💾 Save and organize palettes locally
- 🌐 Share palettes with team members

## Installation
```bash
npm install palette-2
```

## Quick Start
```javascript
import { Palette } from 'palette-2';

const palette = new Palette();
palette.addColor('#FF5733', 'primary');
```
